EXPLODER DECODER

Instructions

There is a bomb. You must defuse it by guessing the correct bomb code before the time runs out. This would be nearly impossible without the DECODER (the little green thing to the right of the bomb). Every time an incorrect code is entered, the decoder will assess each digit that you guessed and display the "direction" of the correct digit. For example, if the correct code is 5555 and you entered 7935, the decoder will display vv^-. "v" means that the correct digit is lower than your guess, "^" means that it is higher, and "-" means that your guess was correct. Good luck!
